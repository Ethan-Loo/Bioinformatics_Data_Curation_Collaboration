This repo contains metadata schema used by the Waldron Lab at CUNY Health Sciences. The schema is intended to validate metadata acquisition and cleaning via LinkML. Collaborators include Arielle Mol, Andy Chen, the the Waldron Lab team.

## Using `LinkML` to curate and validate datasets

In this repo, we utilize [`LinkML`](https://linkml.io/linkml/) to validate curated data and demonstrate LinkML's ability to convert datasets into various formats using a `yaml` schema
